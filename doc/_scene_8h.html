Detailed Description

represents a scene

Author
Jean-Manuel ERIALC - Hadjer DJERROUMI
Version
0.1
Date
11 juin 2020

Contains a type Scene representing a 3D scene and functions related to it. The scene can be seen via a graphic window or exported as an image. The scene contains shapes and light sources.
